# RimTalk-Toddlers-Expansion
RimTalk Expand Toddlers

RimTalk鼠蛋拓展（开发测试中）

1. 更多的家庭互动：（配合toddlers模组）
- 在婴儿1岁后解锁“自娱自乐”娱乐工作，LLM生成工作文本（如玩积木等）
- 在婴儿1岁后解锁“互相玩耍”娱乐工作，LLM生成工作文本
- 重制玩耍的孤独值；
- 仿照原版的“和婴儿玩耍”即“观察成年人工作”，为成年人提供“观看婴儿玩耍”娱乐工作，不占用婴儿；
- 将之前RimTalk和婴儿说话增加娱乐值，降低孤独值的效果移至此处。
- 鼠蛋会摔跤和打架，因此要好好照顾鼠蛋！
- 增加了几种鼠蛋喜欢的玩具，增加了鼠蛋玩玩具的动画
- 增加了鼠蛋的趣味行为（比如偷吃食物）
- 增加了鼠蛋的一些健康状态（比如蛀牙）
- 增加了鼠蛋的心情状态（比如不想看牙医）
- 增加了鼠蛋无聊度机制；

2. 学习语言：（配合toddlers模组）
- 在婴儿1岁后解锁“牙牙学语”健康状态，显示婴儿学说话的进度并传递给LLM，让LLM生成符合婴儿身份的语言；

3. 原版修复：（仅需生物科技DLC）
- 当婴儿和父母在一个房间中时，消除原版的“多人宿舍”负面心理状态；使用“我的宝宝在身边”取代该状态。
- 婴儿现在需要找大人陪着睡觉了，否则会有负面情绪哦
- 修复了Toddlers模组的很多bug

4. 原版趣味：
- 增加了鼠蛋过路者和鼠蛋商队！（会紧紧跟着自己的监护人）
